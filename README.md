# LinkedIn Clone - Backend API

## Live API
https://your-backend.onrender.com

## Technologies
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Environment Variables
```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```

## Local Setup
```bash
npm install
npm run dev
```

## Features
- User Authentication
- Create/Read/Update/Delete Posts
- Like Posts
- Comment on Posts
